# Nanabozho — Branding Guide

## Tone

Handcrafted. Earthy. Clever. Never corporate.

Everden should feel like a river town notice board crossed with a worn field journal — not a SaaS landing page.

## Color Palette (Wetland / Reedwater)

| Role | Hex | Use |
|------|-----|-----|
| Deep marsh | `#1a3c34` | Backgrounds, headers |
| River brown | `#4a3728` | Wood, earth structures |
| Reed green | `#5c7a52` | UI accents, healthy vegetation |
| Mist grey | `#8a9a9e` | Secondary text, fog |
| Lantern amber | `#d4a054` | Interactive highlights, warmth |
| Lily white | `#e8e4d9` | Body text on dark |
| Flood warning | `#8b4513` | Alerts, high water |

Avoid: default blue primaries, purple gradients, glassmorphism cards, Lucide-icon grids.

## Typography

- **Display / titles:** A serif with character (e.g. Lora, Crimson Pro) — worn, readable
- **UI / body:** A humanist sans (e.g. Source Sans 3, Nunito) — cozy, not sterile
- **Combat log / journal:** Monospace optional for dice rolls (e.g. IBM Plex Mono)

## UI Patterns

- Interaction prompts: short verbs ("Talk", "Examine", "Trade")
- Dialogue: species-appropriate speech; no exposition dumps
- Combat log: Baldur's Gate-style scrolling text with dice results visible
- Menus: pause overlay, not modal spam

## Logo / Wordmark (Placeholder)

- Text: **Nanabozho** in display serif, small **presents Everden** beneath
- Optional symbol: stylized ripple or footprint — not a cartoon mascot
- Minimum clear space: height of the "N" on all sides

## What We Avoid

- Humans-in-costume furry aesthetics
- Generic fantasy MMO UI chrome
- Trendy neon cyber palettes
- AI-slop layout (hero → 3 features → testimonials → CTA)

## 2026-07-06 — Launch Species & Combat Roles

**Frogs first, toads, turtles.** More animals sketched over time.

| Species | Combat role | Biology hook |
|---------|-------------|--------------|
| Frog | Mobile / scout | Leap, swim, tongue strike, high initiative |
| Toad | Control / debuff | Poison glands, burrow, intimidation croak |
| Turtle | Tank | Shell block, withdraw, damage reduction |
| Vole (later) | Healer + hide | Burrow stealth, small-frame healing, party sustain |

**Combat feel:** D&D / Baldur's Gate — turn-based, dice rolls, items add multipliers and bonus dice.

**Art:** 2D pixel sprites in 2.5D Three.js world. Nick sketches animals; code handles the rest.
